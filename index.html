Algorithms

Linear Search

In linear search, the idea of the algorithm is to iterate across the array from left to right, searching for a specified element.
O(n) and Ω(1)

Binary Search

In binary search, the idea of the algorithm is essentially to divide and conquer, reducing the search area by half each time to find the target.
O(log) and Ω(1)

Bubble Sort

The idea of the algorithm here is to move higher valued elements towards the right and lower valued elements towards the left.
O(n2) and Ω(n2)

Selection Sort

In selection sort, the idea of the algorithm is to find the smallest unsorted element and add it to the end of the sorted list.
O(n2) and Ω(n2)

Merge Sort

In merge sort, the idea of the algorithm is to sort smaller arrays and then combine those arrays together (hence merge) in sorted order. Merge sort uses recursion.
(n) and Ω(1)
